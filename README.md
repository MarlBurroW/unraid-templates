# Hivekeep Unraid templates

Unraid Community Apps templates for [Hivekeep](https://github.com/MarlBurroW/hivekeep), a self-hosted platform of autonomous, persistent personal AI agents.

## Apps

- **Hivekeep** (`templates/hivekeep.xml`): the Hivekeep platform itself. Web UI on port 3000, all state in a single data volume.
